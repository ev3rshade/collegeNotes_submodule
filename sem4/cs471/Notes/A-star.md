# A-star
an informed search algorithm that uses heuristics to find the lowest-cost path from a start state to a goal state. type of best-first.

Core Evaluation Function combines [[UCS]] with [[Greedy]]
$$f(n) = g(n) + h(n)$$

For tree search: the heuristic must be **admissible**
For graph search: the heuristic must be **consistent**
