# Heuristic
problem specific hint that estimates the coast of the cheapest path from a node $n$ to a goal state.

---
## Characteristics

**Admissible** - never overestimates the actual cost to reach the nearest goal (also non-negative)

> $h(n) \leq h^*(n)$ where $h^*(n)$ is the true cost to reach the goal from node $n$


**Consistent** - estimate is consistent across every "arc' (edge) in the state space

> for every node $n$ and every successor $n'$ generated by an action $\alpha$ heuristic must satisfy the inequality:
> $$h(n) - h(n') \leq cost(n \text{ to } n')$$
