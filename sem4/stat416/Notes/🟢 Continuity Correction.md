# Continuity Correction
a statistical adjustment made when using a continuous distribution (usually the normal distribution) to approximate a discrete distribution (like binomial or Poisson). 

$P(X = n)$  approximated by $P(n - 0.5 < X < n + 0.5)$
$P(X \leq n)$  approximated by $P(X < n + 0.5)$
$P(X \geq n)$  approximated by $P(X > n - 0.5)$
$P(X < n)$  approximated by $P(X < n - 0.5)$
$P(X > n)$  approximated by $P(X > n + 0.5)$
