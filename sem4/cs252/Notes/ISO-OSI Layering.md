# ISO-OSI Layering
Open Systems Interconnection model made by International Organization for Standardization



1 **Application**
individual application programs ([[HTTP]])

2 **Transport**
Program to program ([[TCP]] and [[UDP]])

3 [[Internet]]
packet forwarding, machine to machine ([[IP]])

4 **Network Interface**
Local Area Network (LAN) ([[Ethernet]])

5 **Physical**
Basic Network Hardware
