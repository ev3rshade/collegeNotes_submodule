# DHCP
Dynamic Host Configuration Protocol
-- a network management protocol that automatically assigns:

- IP addresses   | current address
- subnet masks | used to send packets to hosts inside the LAN
- default router | used to send packets to hosts outside the LAN
- DNS servers    | used to convert names to IP addresses

to devices (clients) on a network

before DHCP an admin had to manually configure everything

`ifconfig` - the command to set these parameters
