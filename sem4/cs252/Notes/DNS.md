# DNS
Domain Name Server
a service that translates host names to IP addresses

uses a distributed lookup algorithm and contacts as many servers as necessary

host names are divided in domains:
`host.dom2.dom1.dom0`
e.g. `data.cs.purdue.edu`


before, net admins used to exchange updates in `/etc/hosts` file --> NOT SCALABLE
