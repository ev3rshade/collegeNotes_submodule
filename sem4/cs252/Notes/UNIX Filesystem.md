# UNIX Filesystem

## Important Directories
- Root Directory
- /etc - OS config files
	- /etc/passwd
	- /etc/group
	- /etc/inetd.conf
		- Internet config
		- Daemons - programs running in the background implementing a serviee
	- /etc/rc.\*/
		- OS initialization scripts for different services
- /dev - list of devices attached to the computer
- /usr - libraries and tools
	- /usr/bin - application programs such as grep, ls etc
	- /usr/lib - libraries used by the application programs
	- /usr/include - include files (.h) for the libraries
- /homes - home directories
- /bin - other executables
