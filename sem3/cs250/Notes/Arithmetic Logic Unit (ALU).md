# Arithmetic Logic Unit (ALU)
> The circuit inside the CPU that performs arithmetic (add, subtract) and logical (AND, OR, XOR, shift) operations on binary numbers.

![[Screenshot 2025-12-10 235317.png]]

---

## Function

The ALU is the computational heart of the CPU. It:
- Takes two operands as input (from registers)
- Performs an operation selected by a 4-bit **ALU control signal**
- Produces: **Result**, **Zero flag** (result == 0?), **Overflow flag**

The ALU control signal is generated by the [[Control Unit]] combining the main `ALUOp` signal and the instruction's function field.

---

## ALU Operations

| ALU Control | Operation |
|---|---|
| 0000 | AND |
| 0001 | OR |
| 0010 | Add |
| 0110 | Subtract |
| 0111 | Pass input B (for CBZ) |
| 1100 | NOR |

---

## Scale

A 1-bit ALU slice handles AND, OR, and addition for a single bit pair with carry.

![[Screenshot 2025-12-10 235231 1.png]]

A **64-bit ALU** is built by chaining 64 of these 1-bit slices (ripple-carry) or using a carry-lookahead design for speed.
