# Assembly

> Assembly is an array of instructions that is executed one by one sequentially. Sometimes it jumps to diff instruction locations (branch) tho

C/C++ compilers generates assembly language!!
--> higher level languages will execute on top of programs an libraries compiled in C/C++

![[Screenshot 2026-03-06 232521.png]]

## Tips for Writing Assembly Code

- Write the code in C first
- **WRITE COMMENTS!!!** --> the C code on the side
- Translate common C functions into assembly (e.g. heapsort, adding two numbers, Fibonacci (recursive))
